# c9-front
Frontend para C9

## Instalación
```bash
npm install
```

## Ejecución
```bash
npm run serve
```

## Variables de entorno
```bash
cp .env.example .env
```

dentro del archivo .env se debe configurar la url del backend
```bash
PORT=3001

REACT_APP_BACKEND_URL=http://localhost:3000/api/
```
